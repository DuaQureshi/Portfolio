Personal Portfolio Website

Introduction:

This project is a personal portfolio website created to display skills, services, and contact details in a professional and visually attractive way. It uses a consistent color theme and smooth animations to enhance user experience.

Features:

Fully responsive layout
Animated skill bars and circular skill indicators
Centralized contact section
Dark and elegant color contrast
Clean UI with minimal design
Easy to customize

Technologies:

HTML
CSS
JavaScript

▶️ How to Run the Project:

Download the project
Keep all files in one folder
Open index.html in any modern browser
Navigate through sections using the header menu or scroll.!!

Usage:

This project can be used as:

Personal portfolio
University project
Freelancing profile
Resume website

Note:
No external libraries or APIs are used. The project is lightweight and safe for online hosting without risk of blocking
